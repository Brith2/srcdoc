Task::Wait->new

Task::Wait->new(options...)
Create a new Task::Wait object. The following options are allowed: - All options allowed for Task->new(), except 'mutexes'. - seconds - The number of seconds to wait before marking this task as done or running a subtask. - inGame - Whether this task should only do things when we're logged into the game. If not specified, 0 is assumed. If inGame is set to 1 and we're not logged in, then the time we spent while not being logged in does not count as waiting time.
