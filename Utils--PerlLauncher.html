Utils::PerlLauncher - Abstraction layer for launching Perl scripts

This class is derived from: AppLauncher

This class provides a cross-platform way to launch other Perl scripts. It automatically uses the system's Perl interpreter, or uses (wx)start.exe if that's not available.
