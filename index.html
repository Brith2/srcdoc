The Wx interface

OpenKore has a graphical user interface written with WxWidgets. In short, it's called the Wx interface. It uses many custom controls. Here's a list of some of the documented custom controls.
Interface::Wx::Console Console control.
Interface::Wx::DockNotebook Notebook control which supports undocking of its children
Interface::Wx::DockNotebook::Page Notebook page child used by DockNotebook
