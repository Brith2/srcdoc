Interface - User interface system.

In OpenKore, the user interface code is seperated from the core code. Each user interface is implemented in a class. The Interface class is an abstract base class for all OpenKore user interface classes.

void $interface->errorDialog(String message, [boolean fatal = true])
Parameters:
message : The error message to display.
fatal : Indicate that this is a fatal error (meaning that the application will exit after this dialog is closed). If set, the console interfaces will warn the user that the app is about to exit.
Requires:

Display an error dialog. This function blocks until the user has closed the dialog.

Consider using Log::error() if your message is not a fatal error, because Log::error() does not require any user interaction.

$interface->query

String $interface->query(String message, options...)
Parameters:
message : A message to display when asking for input.
Requires:
defined($message)
Returns:
The user input, or undef if the user cancelled.

Ask the user to enter a one-line input text. The following options are allowed:
  • cancelable - Whether the user is allowed to enter nothing. If this is set to true, then the user will be asked the same thing over and over until he replies with a non-empty input. The default is true.
  • title - A title to display in the query dialog. The default is "Query".
  • isPassword - Whether this query is a password query. The default is false.

$interface->showMenu

int $interface->showMenu(String message, Array<String>* choices, options...)
Parameters:
message : The message to display while asking the user to make a choice.
choices : The possible choices.
for all $k in @{$choices} : defined($k)
Requires:
defined($message) defined($choices)
Ensures:
-1 <= result < @{$choices}
Returns:
The index of the chosen item, or -1 if the user cancelled.

Ask the user to choose an item from a menu of choices.

The following options are allowed:

  • title - The title to display when presenting the choices to the user. The default is 'Menu'.
  • cancelable - Whether the user is allowed to not choose. The default is true.
