Utils::TextReader - UTF-8 text reader.

A convenience class for reading text files encoded in UTF-8. If you're not familiar with UTF-8, Unicode or character encoding in general, then you should read this article by Joel on Software.

If the text file is not valid UTF-8, then it will assume that the text file is in the system's default encoding. If that isn't correct either, then an exception will be thrown during reading.

This class is to be used as follows: 

use Utils::TextReader;

my $reader = new Utils::TextReader("file.txt");
while (!$reader->eof()) {
   print $reader->readLine();
}
