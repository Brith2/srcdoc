Task::Chained - Easy chaining of different tasks.

This task allows you to easily combine different tasks into a single, more complex task. Given a list of tasks, Task::Chain will execute all of those tasks, in the same order as they are given. Task::Chain will stop when all tasks are finished, or if a task failed.

Task::Chained will also use the same mutex as the currently active subtask.

Example

The following example creates a Task::Chained task which first stands, then waits 3 seconds, and then sits again. 
my $task = new Task::Chained(
    tasks => [
        new Task::SitStand(mode => 'stand'),
        new Task::Wait(seconds => 3),
        new Task::SitStand(mode => 'sit')
    ]
);
